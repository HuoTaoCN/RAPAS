# GovInsight-AI RA 重复诉求智能分析系统

`GovInsight-AI RA` 是 `GovInsight-AI` 系列中的重复诉求智能分析子系统，面向政务热线与运营管理场景，用于识别同一诉求人在一段时间内的重复诉求模式、内容关联性与处置优先级。

## 当前原型包含

- 分析工作台
- 结构化 / 非结构化输入示例
- 分析总览指标区
- 报告预览区
- 时间链路视图
- 分级处置建议
- 风险与审慎提示

## 系列标识

- 系统名称：`GovInsight-AI RA 重复诉求智能分析系统`
- 系列域名：`ra.huotao.com`
- 英文名称：`Repetitive Appeal Pattern Analysis System`

## 说明

当前仓库内容以静态原型为主，用于界面设计、信息架构展示和后续产品开发对齐。系统分析结论仅作为辅助参考，最终处置决定需由人工审核确认。
